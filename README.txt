# Memory Palace Trainer PWA

## 使い方

1. index.html をブラウザで開く
2. または GitHub Pages にアップロード
3. スマホで開き「ホーム画面に追加」
4. pcでの更新されない問題の解決。キャッシュ保存される為にダウンロードした場合それを削除
　　対象サイトの上のアドレスバーの左側からクッキーとサイトデータの項目からデバイス上のサイトデータを管理から消す。
　　それでCtrl + Shift + R で強制再読み込み。これでpcの方はいけるはず。
5.アンドロイドアプリの場合も同じく対象サイトの上のアドレスバーの左側からクッキーとサイトデータの項目から保存データを消す
それで対象アドレスページを更新する。

これは上げる側で使う側には関係ないけど上げる場合、htmlの更新だけでは更新されない。一緒にservice-worker.jsの方もv3v4と更新していかないと反映されないらしい。のでここは注意

## ファイル説明

- index.html … メインアプリ
- manifest.json … PWA設定
- service-worker.js … オフライン対応
- icon-192.png / icon-512.png … アプリアイコン
